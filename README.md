### Installation and Setup Instructions

```bash
clone the project
git clone https://github.com/TasfiaIslam/Tutorhub.git
```
- Backend
```bash
cd backend
npm run develop

#Then visit http://localhost:1337/admin to view the backend.
```

- Frontend 
```bash
cd frontend
npm run dev

#Then visit http://localhost:3000 to view the app.
```
